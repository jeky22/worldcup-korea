# Product

## Register

product

## Users

- **한국 축구 팬**: 경기 중·직후 스마트폰으로 A조 순위, 다음 경기, 진출 시나리오를 빠르게 확인하려는 사용자
- **일반 축구 팬**: 48팀·104경기 전체 일정, 결과, 팀·선수 정보를 한곳에서 탐색하려는 사용자
- **공통 맥락**: 대회 기간(2026.6–7) 중 실시간에 가깝게 정보를 소비. 여러 앱·사이트를 오가며 파편화된 정보를 찾는 불편을 해소하는 것이 핵심 동기

## Product Purpose

2026 FIFA 월드컵 정보 허브. 세 가지 가치를 동등하게 제공한다:

1. **경기 정보** — 일정, 스코어, 경기장, 조별 순위를 한 화면에서
2. **팀·선수** — 스쿼드, 라인업, Transfermarkt 기반 연봉(추정치, 출처 표기)
3. **진출 시나리오** — 대한민국(A조) 기준 남은 경기 조합 시뮬레이션(FIFA 2026 타이브레이커 적용)

**데이터 원칙**: 더미·목업 데이터 금지. openfootball, wcup2026, Wikipedia(MediaWiki API), Transfermarkt 등 실제 외부 소스만 사용. API 키 불필요. 동기화 실패 시 빈/에러 상태 — 가짜 값으로 채우지 않음.

**성공 기준**: 사용자가 "체코 vs 멕시코가 어떻게 되면 한국이 올라가는지"를 30초 안에 이해하고, 경기·선수 정보를 별도 검색 없이 확인할 수 있다.

## Brand Personality

**날카롭고 · 정확하고 · 데이터 중심**

- 숫자와 표가 주인공. 장식·감성 카피는 최소
- 출처·갱신 시각을 항상 노출해 신뢰 확보
- 대회 에너지는 **데이터의 생동감**(라이브 스코어, 순위 변화)으로 전달 — 배경 그라데이션이나 장식적 비주얼이 아님

## Anti-references

- **SaaS 대시보드**: 회색 카드 그리드, 히어로 메트릭(큰 숫자 + 작은 라벨), 아이콘+제목+설명 반복 카드
- **뉴스 포털**: 광고 밀집, 기사 나열형 레이아웃
- **게임 UI**: 네온, 과한 그라데이션, 글래스모피즘 장식
- **AI 슬롭**: 크림/베이지 바디 배경, 섹션마다 작은 대문자 eyebrow, 그라데이션 텍스트, 좌측 컬러 스트라이프

## Design Principles

1. **데이터가 디자인이다** — 시각적 장식보다 정보 밀도·가독성·정확성 우선
2. **출처를 숨기지 않는다** — 모든 데이터 블록에 소스·갱신 시각 표기(연봉은 비공식 추정치임을 명시)
3. **한국 시나리오가 차별점** — 전체 대회 정보는 균등 제공하되, A조 진출 계산기는 핵심 기능으로 대우
4. **경기 중 모바일** — 엄지 한 손으로 순위·다음 경기·시나리오 결과 확인 가능한 터치 타깃·레이아웃
5. **실패는 솔직하게** — 데이터 로드 실패 시 스켈레톤/가짜 데이터 대신 명확한 에러와 재시도

## Accessibility & Inclusion

- **WCAG 2.1 AA** 준수(본문 대비 ≥4.5:1, 포커스 표시, 시맨틱 마크업)
- **모바일 웹 우선** — 경기 중 스마트폰 사용이 주 사용 환경
- **의도적 모션** — 라이브 스코어·순위 변화에 생동감 있는 전환 허용. 단, `prefers-reduced-motion: reduce` 시 즉시 대체(크로스페이드 또는 즉시 전환)
- **색맹 고려** — 순위·진출 상태는 색상만으로 구분하지 않고 아이콘·라벨 병행
